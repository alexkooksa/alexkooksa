Restte

Online furniture store in HoReCa segment

  • Design
  • Front-end development
  • CMS development
2021

Goals

Develop a standalone website for HoReCa segment for a regular client who owns an online furniture and decor shop. The website should have an elegant design, clean layout, and standard features.

Page types

  1. Homepage
  2. Products list
  3. Product page
  4. Search results
  5. Favorites
  6. Cart and checkout
  7. Blog
  8. Static page
  9. Personal account
  10. Contacts

Homepage

The main page has a classic block of banners that the client can manage from the admin area. Below this, there is a navigation to the main categories: furniture for restaurants, furniture for offices and furniture for hotels. The section of interior examples allows the customer to be inspired by how the product is used in practice.

Products list

At the top of the screen there is a typical cover block: H1 title consisting of the name of the main category and subcategory, a short description and a stylish image. Inside the catalogue itself, there are a number of filters available for product selection.

Product page

All basic information about the product is available, as well as the opportunity to submit a request for a complete selection of furniture for the project. Some products have a restriction in the form of a minimum quantity for ordering, we immediately warn the customer about it.

Result

  • The site was developed using an in-house CMS, including a special module to handle shop-related features. Front-end: HTML + Smarty template engine, CSS, JQuery + plugins. Back-end: PHP, CodeIgniter.
  • The well-designed catalogue hierarchy had a positive effect on SEO.
  • Easy to use even on small devices.
  • Content relevance is maintained by client moderators through the use of an administrative panel and synchronization.
