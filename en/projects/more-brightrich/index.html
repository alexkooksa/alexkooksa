The company

Bright Rich is a member of CORFAC International, an international association of professional brokerage companies. As of today, their portfolio of transactions is higher than 1.9 million sq. m. The branches are located in Dubai and Almaty, Moscow and St. Petersburg.

The goals

Besides the main website migration (which you can read about in this case study), many other projects were completed. Here are some of them.

BR Home

  • Front-end development

Goals

Bright Rich has started a new line of business - the luxury housing segment. We are here to develop a landing page as soon as possible.

Process

The work was done in collaboration with the client's designer. The result was a minimalist one-page with subtle animations.

Result

The project was completed in 3 days, to the customer's great delight. Further development of the structure is planned. The result successfully passed the designer's review. The site is fully responsive and displays perfectly on small screens.

Moscow City Broker

  • UI/UX audit
  • Redesign
  • Coding and CMS integration

Goals

Make a slight redesign and enhance the user experience of the current website. Rebuild the website from scratch using Evernet's CMS designed for the property industry.

Process

We identified and documented the issues found during the audit. Then the design layouts were approved by the client and brought to life. The tech stack was classical (HTML, CSS, jQuery + plugins, Smarty templates). We also implemented specific SEO techniques to enhance the website's organic rankings.

Problem

Sloppy layout in the filter, poorly read elements in the header, missing title

Solution

The colour of the header has been changed to white, an H1 header has been added and the layout of the filter has been improved.

Problem

Inadequate scrolling buttons. The block was lacking any features other than switching to the chosen tower.

Solution

An information area is added below this slider. This section provides quick access to information about the currently selected tower while scrolling.

Problem

The contact form is not distinguishable from the rest of the content.

Solution

Added call to action and fields to specify the request.

Problem

The towers list page was missing.

😥

Solution

The page displaying all the business complex towers has been developed.

Problem

There was no slider to show photos, instead of actual photos of the premises, a photo of the business centre was displayed

Solution

Slider added. Enquiry form pinned to the right. Offer page was brought to the consistent style.

Final result

A SEO-friendly site with many improvements compared to the previous version. It ranks well and displays correctly on all screen sizes in different browsers.

BR Cash

  • Design
  • Front-end development

Goals

Develop a personal account for the Bright Rich affiliate programme for private agents and brokers.

Result

The deals are displayed in blocks with a progress bar and are categorised by property type using specific colours. It is possible to contact a manager for each deal. After closure, relevant documents can be downloaded. The sales funnel sidebar remains visible at all times and can be collapsed if necessary. The project has been launched and is actively used by brokers.

BR CRM

  • Design
  • Front-end development

Goals

Develop a system (CRM) for the internal usage of Bright Rich brokers.

Process

The standard CRM layout with sidebar was used as the basis. The interface design follows the typical Bright Rich style.The primary section displays a list of offers, which can be filtered. Brokers are allowed to create selections, while moderators can add and modify items.

Result

The Evernet team runs and maintains the system, which is built with the classic stack featuring HTML, SCSS, jQuery and Smarty templates for component reuse. PHP is used on the backend for efficient functionality.
