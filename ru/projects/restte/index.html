Restte

Интернет-магазин мебели для сегмента HoReCa

  • Дизайн
  • Вёрстка и интеграция с CMS
2021

Задача

Разработать отдельный сайт по направлению HoReCa для постоянного клиента, владеющего интернет-магазином мебели и декора. Сайт должен быть стильным и лаконичным и иметь стандартный функционал.

Типы страниц для вёрстки

  1. Главная страница
  2. Категория товаров
  3. Страница товара
  4. Результаты поиска
  5. Избранное
  6. Корзина
  7. Блог
  8. Статичная страница
  9. Личный кабинет
  10. Контакты

Главная страница

На главной разместился классический блок с баннерами, которыми клиент может управлять из админки. Ниже сделан переход в основные категории: мебель для ресторанов, офисов или гостиниц. В блоке с примерами мебели в интерьере покупатель может вдохновиться использованием товара в реальной жизни.

Категория товаров

В верхней части экрана сделан типовой блок-обложка: заголовок H1, состоящий из названия основной категории и подкатегории, короткого описания и стильной картинки. В самом каталоге доступно множество фильтров для подбора товара.

Страница товара

Доступна вся основная информация о товаре, а также возможность оставить заявку на полный подбор мебели для проекта. У некоторых товаров есть ограничение в виде минимального количества для заказа, об этом сразу предупреждаем покупателя.

Результат

  • Сайт сверстан на собственной CMS с модулем для магазина.
  • Front-end: HTML + шаблонизатор Smarty, CSS, JQuery + plugins. Back-end: PHP, CodeIgniter.
  • Продуманная иерархия каталога положительно отразилась на SEO.
  • Удобно пользоваться в том числе и на небольших устройствах.
  • Актуальность контента поддерживается модераторами клиента через админ-панель и синхронизацию.
