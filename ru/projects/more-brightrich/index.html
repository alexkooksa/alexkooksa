О компании

Bright Rich — первая российская компания, принятая в CORFAC, международную ассоциацию брокеров коммерческой недвижимости. На сегодня портфель сделок превышает 1,9 млн кв. м. Офисы открыты в Москве, Санкт-Петербурге, Дубае и Алматы.

Задачи

Помимо переноса основного сайта (о котором можно прочитать в этом кейсе) для компании был выполнено множество других проектов. Вот некоторые из них.

BR Home

  • Вёрстка

Задача

Компания Bright Rich запустила новое направление — сегмент элитного жилья. Нужно разработать имиджевый сайт в кратчайшие сроки.

Процесс

Работа велась совместно с дизайнером клиента. По макетам из Фигмы был свёрстан минималистичный одностраничник с небольшим количеством анимаций.

Результат

Проект был запущен за 3 дня, что очень порадовало клиента. В дальнейшем планируется расширение структуры. Верстка успешно прошла авторский надзор дизайнера. Сайт адаптивен и отлично отображается даже на небольших экранах в различных браузерах.

Moscow City Broker

  • Аудит
  • Редизайн
  • Вёрстка и интеграция с CMS

Задача

Сделать небольшой редизайн и улучшить UX на имеющемся сайте. Сверстать с нуля на собственной CMS Evernet, заточенной под работу с недвижимостью.

Процесс

Провели аудит и составили список багов и неудобств (примеры далее). Нарисовали макеты и согласовали их с клиентом. Верстка интегрирована с CMS. Использован стандартный набор технологий: HTML, CSS, JQuery + плагины, шаблонизатор Smarty. Провели ряд работ по SEO для поднятия сайта в органической выдаче.

Проблема

Неаккуратная вёрстка в фильтре, плохо считываются элементы в шапке, отсутствует заголовок

Решение

Цвет шапки заменен на белый, добавлен заголовок H1, улучшена вёрстка в фильтре

Проблема

Недостаточно заметные кнопки для прокрутки, отсутствие функционала кроме перехода в выбранную башню

Решение

Добавлен информативный блок под слайдер. Теперь при прокрутке сразу видна информация об активной башне

Проблема

Форма для связи никак не зонирована

Решение

Добавлен призыв к действию и поля для конкретизации запроса

Проблема

Отсутствовала страница с листингом башен

😥

Решение

Сверстана страница со всеми башнями делового комплекса

Проблема

Не был предусмотрен слайдер для показа фото, вместо реальных фото помещений отображалось фото БЦ

Решение

Добавлен слайдер, форма запроса закреплена справа, страница предложения приведена к общему стилю.

Итоговый результат

Получился SEO-friendly сайт со множеством улучшений, в сравнении с существовавшей версией. Отлично ранжируется и корректно отображается на всех размерах экранов в различных браузерах.

BR CRM

  • Дизайн
  • Вёрстка

Задача

Разработать систему (CRM) для внутреннего использования брокерами Bright Rich.

Процесс

За основу взят классический вид CRM с боковым меню. Дизайн элементов интерфейса в привычной стилистике Bright Rich. Основной раздел — листинг предложений с фильтрами. Брокеры могут создавать подборки, а модераторы добавлять и редактировать объекты.

Результат

Система сверстана с использованием классического стека (HTML, SCSS, JQuery, шаблоны Smarty для переиспользования компонентов, PHP на бэкенде), запущена и поддерживается командой Evernet.
